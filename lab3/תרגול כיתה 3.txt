תרגול כיתה 3
תכנות מונחה עצמים


בתרגיל בית 2 השאלה הראשונה היא לממש עץ - שהוא מבנה נתונים רקורסיבי. לכן כדאי לתרגל יצירת מבני נתונים כאלה.


בתרגיל זה נכתוב מחלקה בשם List המממשת רשימה דו-כיוונית של מספרים שלמים. לשם כך נגדיר מחלקה Node שהמופעים שלה ישמשו צמתים של הרשימה המקושרת. הנה השיטות הפומביות של המחלקה List:


public void addAtHead(int val)
	הוספת val בראש הרשימה.
	public void addAtTail(int val)
	הוספת val בסוף הרשימה.
	public int deleteHead()
	מוחקת את ראש הרשימה ומחזירה את הערך בה.
	public int deleteTail()
	מוחקת סוף הרשימה ומחזירה את הערך שם.
	public int deleteAt(int i)
	מוחקת את האיבר במיקום i ומחזיר את הערך בו.
	



זהו קישור לתיקייה בה מופיע הפרויקט כולו.


נקודות ששווה לשים לב אליהן:
1. המחלקות Node ו-List נמצאות בחבילה משלהן list, בעוד המחלקה עם ה-main שמשתמש בהן נמצאת בחבילה אחרת (פה פשוט ב-default).
2. לכן יש שורת import בראש המחלקה של ה-main.
3. כיון ש-Node לא אמור להיות חשוף מחוץ לחבילה, אז המחלקה עצמה, וכך גם השיטות שלה הן לא public, אלא דווקא package-private. 
4. כרגיל, השדות שם הם כן private.


כמובן שיש עוד המון דרכים לממש את השיטות האלה. הבחירה בפתרון הנתון היא לתת למחלקה Node לטפל בכל הקשרים בין Node-ים, ולמחלקה List לטפל ב-head וב-tail.
